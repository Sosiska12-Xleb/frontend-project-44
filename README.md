ПРОЕКТ "ИГРЫ РАЗУМА (JS)"

Проект "Игры разума" - это программа состоящая из 5-ти проверяющие мышление, логику и математические знания. Проект состоит из следующих игр: "Проверка четности", "Калькулятор", "НОД", "Арифметическая прогрессия", "Простое ли число?".

Требования:
Версия Node.js: v24.14.0
Библеотеки: readline-sync

Установка и запуск:
После копирования проекта с github на ваше устройство, нужно установить все нужные библеотеки указанные выше или в файле package.json. После установки можно начинать игру!
Команды для запуска игр (Команды запускаются в консоли):
brain-even - "Проверка четности"
brain-calc - "Калькулятор"
brain-gcd - "НОД"
brain-progression - "Арифметическая прогрессия"
brain-prime - "Простое ли число?"

Возможные ошибки:
Если игры не запускаются выполните команду make link.
